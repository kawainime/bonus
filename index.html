1. Persiapan Awal (Setup)

Sebelum menjalankan tools, pastikan sistem Anda sudah memiliki:

  • Python 3.10+ terinstall di komputer.
  • Koneksi internet yang stabil (disarankan menggunakan Proxy/VPN jika IP terkena limit).

Instalasi Library

Buka terminal/CMD di folder utama script ini, lalu jalankan perintah berikut untuk menginstall semua kebutuhan:

pip install -r requirements.txt

*Pastikan file requirements.txt sudah berisi: requests, httpx, Pillow, cloudscraper, colorama.

2. Cara Menjalankan (Main Launcher)

3. Konfigurasi Khusus (PENTING)

Beberapa tool memerlukan setting tambahan agar bisa berjalan, terutama Veterans Tool.

Setting Veterans Tool (Menu 2)

Tool ini menggunakan API ChatGPT untuk verifikasi. Anda wajib mengedit file konfigurasi:

  1. Masuk ke folder veterans-verify-tool.
  2. Rename config.example.json menjadi config.json.
  3. Buka config.json dengan Notepad/VSCode dan isi:
{
  "accessToken": "EYJhbGciOi...", // Ambil dari https://chatgpt.com/api/auth/session
  "programId": "690415d58971e73ca187d8c9",
  "email": {
    "email_address": "emailmu@gmail.com",
    "email_password": "app_password_disini", // Gunakan App Password jika Gmail
    "imap_server": "imap.gmail.com"
  }
}

Selain itu, isi data target di file data.txt (copy dari data.example.txt) dengan format:

NamaDepan|NamaBelakang|Branch|TanggalLahir|TanggalDischarge

4. Troubleshooting & Tips

  • Missing Module Error: Jika muncul error ModuleNotFoundError, pastikan Anda sudah menjalankan pip install -r requirements.txt.
  • IP Limit / Block: Jika verifikasi gagal terus menerus, gunakan Proxy. Buat file proxy.txt di dalam folder tool terkait (format: user:pass@host:port).
  • Google Chrome Error: Untuk tool yang menggunakan browser otomatis, pastikan Google Chrome terinstall di PC Anda.
